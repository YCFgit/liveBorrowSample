机器人消息测试

批量发送人与机器人会话中消息接口需要 `robotCode + userIds + msgKey + msgParam`；是否能真实送达，还取决于机器人能力、消息模版和应用可见范围是否已在钉钉后台配置。

互动卡片测试

互动卡片是否能真实发送，取决于你是否已在钉钉后台创建卡片模版、开启卡片能力，并将示例中的 `cardTemplateId`、`outTrackId`、接收人参数替换成真实值。
